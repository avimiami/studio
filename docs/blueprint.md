# **App Name**: Velocity Chaser

## Core Features:

- Car Selection: Car selection page where users select car color of race car
- Race Display: Main game component to run/render race
- Score display: Score counter displayed while game is played.
- Game logic: Basic game logic implementation. User controls the 'Bugatti' car around a race track as 'Chevy' pace cars give chase.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to evoke a sense of speed and modernity.
- Background color: Light gray (#E9ECEF), creating a clean and neutral backdrop for the vibrant elements.
- Accent color: Bright orange (#FF8C00), providing a high-energy contrast to the primary blue.
- Headline font: 'Space Grotesk' sans-serif for a computerized, techy feel. Body text: 'Inter' sans-serif with a modern, machined, objective, neutral look. Use 'Space Grotesk' for headlines and 'Inter' for body.
- Simple icons that reflect the car's direction of travel.
- Clean layout with car selection and race as the main view. Score at top.
- Smooth animations for car movements and transitions.
